# RLSS Swimmer Profile Site

This folder is ready to upload to GitHub as a static site.

Files included:
- `index.html`
- `style.css`
- `script.js`
- `results.json`

How to use:
1. Upload the contents of this folder to your GitHub repository.
2. Enable GitHub Pages for that repository.
3. Keep all four files in the same directory.

The site uses the cleaned swimmer profile dataset with these fields:
- `year`
- `name`
- `event`
- `age_group`
- `final_time`
- `club`

The JSON has already been cleaned to remove split times, duplicate open-age rows, and most non-individual entries.
